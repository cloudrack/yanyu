git的使用

git命令

  1. linux常用命令

  2. ls 显示列表

  3. mv 移动文件
  4. mkdir 创建文件夹
  5. rm 删除文件
  6. rm -r删除目录
  7. cd 进入目录

  8. clear 清屏

  9. git操作命令

  10. git init 初始化git仓库

  11. git clone [url] 克隆远程服务器上的仓库到本地
  12. git add . 将当前文件添加到暂存区
  13. git status 查看文件状态
  14. git commit -m "msg" 将暂存区内容提交到本地仓库
  15. git push 提交到远程仓库（通过clone到本地后可以操作）

  16. git remote add origin https://git.com/email/xxx.git命令上传到gitee仓库

  17. 分支操作命令

  18. git branch 查看分支

  19. git branch -r 查看远程仓库分支
  20. git branch dev 创建分支
  21. git checkout -b [branch] 切换分支
  22. git merge [branch] 合并到指定分支
  23. git checkout -d [branch] 删除分支

gitee设置中添加上传代码电脑的公钥.SHH，使用命令（C:\Users\admin>下）ssh-keygen -t rsa -C "xxx@xxx.com"创建公钥，按三次回车才能创建。创建后在user/admin中找到.ssh文件夹，找到公钥文件。

配置命令

  • $ git config --global --list 查看当前用户配置
admin@LAPTOP-0S331G62 MINGW64 /d/yunPan/test
$ git config --global --list
user.name=crk
user.email=crk231@163.com
  • git config --global user.name "crk"
  • git config --global user.email "crk231@163.com"
admin@LAPTOP-0S331G62 MINGW64 ~/Desktop
$ git config --global  user.name "crk"
admin@LAPTOP-0S331G62 MINGW64 ~/Desktop
$ git config --global user.email "crk231@163.com"

本质是计算机当前用户的config配置文件信息
